# 準備

`a.cpp`というファイルを作成して，次のコードを書く．

```cpp
#include<iostream>
using namespace std;
int main(){
  cout<<"Hello World"<<'\n';
}

```

コンパイルして，実行する．
```bash
$ g++ a.cpp

$ ./a.out
Hello World

$ 
```
